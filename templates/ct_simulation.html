CT Simulation

Initial Object to Scan

The goal of this project is to demonstrate how a CT Scan creates an image. Rather than buying a CT Scanner for this project, we can generate a similar dataset to the dataset that the CT Scanner generates. Let's start with a known cross-section.

Sinogram Creation (Basic)

A CT Scan generated a pattern known as a sinogram. That is, that objects in a cross-section are represented by a a series of sine waves. Notice in this image it almost makes sense what you are looking at. The Structures in the original image correspond to shapes in the sinogram.

Sinogram Creation (Advanced)

Now the same method is done on a real cross-section. Now the sinogram is very complicated. However, we know there is a mathematical formulation for getting back to the original image. We know this because there is a relationship between objects in the original cross-section and the sinogram (see above).

CT Simulation

Sinogram (Single Row)

This is a single row of a sinogram, there is a spacial dimension on the x-axis and an intensity dimension on the y-axis. These values are then assigned a relative brightness and form one column of the sinogram. You can think of a sinogram as a stack of these plots.

Final Simulation

After adding all the values obtained from the single rows of sinograms, now the final sinogram is obtained. This is the final output from the simulation.
