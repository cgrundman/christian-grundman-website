# Personal Website

Personal website

## Run Locally

Clone the project:

```bash
  git clone https://github.com/cgrundman/christian-grundman-website
```

Go to the project directory:

```bash
  cd local/path/christian-grundman-website
```

In the terminal:

```bash
  npm start
```
